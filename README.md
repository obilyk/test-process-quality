# test-process-quality

Репозиторій **test-process-quality** містить три програмні системи, розроблені в межах дослідження якості процесу тестування програмного забезпечення. Програми призначені для підтримки прийняття рішень у тестовому процесі та автоматизації аналітичних задач.

---

## Призначення репозиторію
Репозиторій є частиною дослідження з оцінювання якості процесу тестування програмного забезпечення та демонструє практичну реалізацію:

1. Методу визначення вагових коефіцієнтів факторів якості.
2. Інтелектуальної системи пошуку дублікатів звітів про помилки.
2. Cистеми автоматичного формування оптимального розкладу виконання тестових сценаріїв між тестувальниками.
