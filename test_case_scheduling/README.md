## Програма формування розкладу тестувальників  
**Метод:** дискретно-часова MILP-оптимізація

### Опис
Програма призначена для автоматичного формування оптимального розкладу виконання тестових сценаріїв між тестувальниками з урахуванням їх індивідуальної доступності та обмежень.

З метою врахування цих факторів запропоновано **дискретно-часову MILP-модель** (Mixed-Integer Linear Programming), яка забезпечує оптимізацію розкладу тестових сценаріїв з урахуванням персоналізованих обмежень доступності тестувальників.

### Реалізація
Модель реалізовано мовою програмування **Python** із використанням оптимізаційного інтерфейсу **PuLP**.  
Для розв’язання задачі застосовується обчислювальний модуль **CBC**.

### Основні можливості
- Автоматичне призначення тестових сценаріїв тестувальникам.
- Урахування:
  - тривалості тестів;
  - часових вікон доступності;
  - сумісності тестів і виконавців;
  - фіксованих подій (мітинги, перерви).
- Мінімізація часу завершення всього набору тестів.
- Формування оптимального розкладу на робочий день.
